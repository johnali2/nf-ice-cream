# nf-ice-cream
A simple Ice Cream store

## Task

Create a simple webpage that looks like the image using HTML and CSS.

-  Home should take the user to the main page.
-  About should not redirect anywhere
-  Shop should take the user to the flavours section
-  Log In is a button but it does not do anything.

The HTML file should be named index.html.

The CSS should live in its own file called style.css linked to the HTML.

Fork this repository. Commit your changes and push to your repo.
